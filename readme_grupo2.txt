O Presente programa foi desenvolvido em python e from utilizadas diversas bibliotecas para IA (ex. bibliotecas para árvores de decisão).
Para utilizar o programa deve ter instalado 'ANACONDA NAVIGATOR 3', pois os mesmo possui as bibliotecas usadas no desenvolvimento.

1-Após instalar 'ANACONDA NAVIGATOR 3':
2-Abra o 'Spyder' (um modulo do anaconda).
3-Clique em File->Open
4-Abra o ficheiro (onde estiver localizado) com o nome 'malaria.py', e execute-o (o mesmo faz o tratamentos dos dados com uma árvore de decisão).
5-Após abrir o ficheiro citado acima, abra e execute outro ficheiro com o nome 'teste.py'(permite fazer um teste de malária em função dos dados).
6-Introduza os dados para poder determinar o resultado.
7-Observe os resultados.
